## Feature ideas

- **No-decisions mode** — after setup, "Chowder, take care of dinner" produces meal plan + recipes + shopping list + fulfillment guidance with no further input required
- **Guided mode** — selective participation: "Give me three options", "Use up the zucchini", "Swap Tuesday's meal", "Two vegetarian dinners"
- **Power user mode** — edit canonical data, tune preferences, adjust novelty thresholds, select AI models, customize prompts, choose fulfillment providers, review all calculations
- **Voice cooking companion** — step-by-step cooking via Siri: "What's next?", "Repeat that step", "How much cumin did we use?"
- **Shareable links** — share recipes, meal plans, and cooking instructions without requiring Chowder to be installed
- **Fulfillment automation** — "Chowder, order everything" via grocery delivery APIs; cart generation; one-tap ordering
- **Image input** — photograph a meal or ingredient label to import into Chowder
- **Recipe URL ingestion** — paste any recipe URL; Chowder normalizes to canonical portable format
